# Citi Invest - Loan Platform with Stripe

## Branding
Colors extracted from logo: Citi Blue #1D4ED8, Light Blue #BFDBFE, Accent Red #EF4444, Dark Blue #1E3A8A

## Setup
1. `npm install`
2. Add Stripe keys to .env
3. `npm start`
4. Login: admin@citiinvest.com / admin123
